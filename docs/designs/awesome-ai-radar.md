## Vision

### 10x Check
awesome-ai-radar becomes the **intelligence layer for the entire Claude Code ecosystem**. Not just a list you read — a living system that connects trends to YOUR code. The skill reads your CLAUDE.md, scans your dependencies, and contextualizes every finding to your specific projects. The public site is the broadcast layer. The skill is the personalized intelligence layer. The repo structure is a **pattern, not just a product** — forkable for any domain (web-radar, security-radar, devops-radar).

### Platonic Ideal
Monday morning. You open your terminal. Before you type anything, your Claude Code session greets you: "3 things happened in AI this weekend that affect your work." One is a new MCP server that replaces a workaround in your project. One is a breaking change in a library you use. One is a cheaper model for your API calls. Each has a one-paragraph "why you should care" and a suggested action. Meanwhile, the public site shows a radar visualization with community-validated signal ratings, trend movements, and deep-dives.
