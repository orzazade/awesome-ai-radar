## What happened

Claw Code launched on April 2, 2026 as an open-source AI coding agent framework built in Python and Rust, described as a clean-room rewrite of the Claude Code architecture. The project achieved 72,000 GitHub stars and 72,600 forks within its first days, making it one of the fastest-growing open-source repositories in the AI tooling category. It provides multi-agent orchestration, tool-calling, and terminal-native AI development through a fully open and extensible harness architecture.

## Why it matters

The demand for open, inspectable AI coding agent harnesses is clearly enormous — 72K stars in days is a strong signal. While proprietary tools like Claude Code, Codex, and Cursor dominate the market, Claw Code offers developers a fully open alternative they can study, fork, and extend. The clean-room design (no proprietary code or third-party model weights) means it can be freely adapted for enterprise and custom deployments.
